VLAN

VLAN (Virtual Local Area Network) adalah teknologi jaringan yang membagi satu jaringan fisik menjadi beberapa jaringan logis yang lebih kecil dan terpisah, memungkinkan isolasi lalu lintas data untuk meningkatkan keamanan dan kinerja, serta memudahkan pengelolaan dan segmentasi jaringan tanpa harus mengubah struktur fisik atau menambah kabel baru

Konfigurasi Router

Router


Router> enable
Router# configure terminal

! Subinterface VLAN 10
Router(config)# interface fa0/1.10
Router(config-subif)# encapsulation dot1Q 10
Router(config-subif)# ip address 192.168.10.1 255.255.255.0
Router(config-subif)# no shutdown
Router(config-subif)# exit

! Subinterface VLAN 20
Router(config)# interface fa0/1.20
Router(config-subif)# encapsulation dot1Q 20
Router(config-subif)# ip address 192.168.20.1 255.255.255.0
Router(config-subif)# no shutdown
Router(config-subif)# exit

! Subinterface VLAN 30
Router(config)# interface fa0/1.30
Router(config-subif)# encapsulation dot1Q 30
Router(config-subif)# ip address 192.168.30.1 255.255.255.0
Router(config-subif)# no shutdown
Router(config-subif)# exit

Router(config)# exit
Router# write memory


Konfigurasi Switch

Switch


Switch> enable
Switch# configure terminal

! Buat VLAN
Switch(config)# vlan 10
Switch(config-vlan)# name IT
Switch(config-vlan)# exit
Switch(config)# vlan 20
Switch(config-vlan)# name HR
Switch(config-vlan)# exit
Switch(config)# vlan 30
Switch(config-vlan)# name FINANCE
Switch(config-vlan)# exit

! Atur port access ke VLAN masing-masing
! Misal PC1 ke FastEthernet0/1, VLAN 10
Switch(config)# interface fastEthernet0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10
Switch(config-if)# exit

! PC2 ke FastEthernet0/2, VLAN 20
Switch(config)# interface fastEthernet0/2
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 20
Switch(config-if)# exit

! PC3 ke FastEthernet0/3, VLAN 30
Switch(config)# interface fastEthernet0/3
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 30
Switch(config-if)# exit

! Set port trunk ke router (Router-on-a-Stick)
Switch(config)# interface fastEthernet0/24
Switch(config-if)# switchport mode trunk
Switch(config-if)# exit

Switch(config)# exit
Switch# write memory
